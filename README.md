# Biblioteca

PW - Biblioteca

## Descripción

Este proyecto tiene como objetivo desarrollar un sistema de biblioteca digital que permita a los usuarios acceder a libros de forma organizada.

## Propósito

- Facilitar el acceso a información de libros.
- Permitir que los usuarios agreguen libros y se involucren
  en la gestión de los mismos.
- Presentar los libros de forma simple.
- Fomentar la interacción mediante filtros de búsqueda y sistema de votos.

## Alcance

El sistema contempla:

- Registro y autenticación de usuarios.
- Subida de libros y datos asociados.
- Visualización en tarjetas con información básica.
- Filtros de búsqueda por coincidencia, categorías y etiquetas.
- Sistema de votos y calificaciones.

## Módulos del Sistema

1. **Gestión de usuarios**: registro, login, perfiles y roles.
2. **Carga de libros**: subida de PDF y sus datos.
3. **Visualización en tarjetas**: presentación de libros con datos básicos.
4. **Filtros y búsqueda**: coincidencia de texto, categorías, etiquetas.
5. **Sistema de votos**: calificación y ordenamiento por relevancia.
